Class JsonPackingConverter

    PackingConverter

    public class JsonPackingConverter
extends AbstractTextConverter
    Converts ImagePacking to a JSON(P) file, with the result that each icon's details are available via a named property which encapsulates w, h, x, y and xy as pixel values as well as storing integer values for x, y, w and h in the property n.
    Author:
    Robert Murphy, mklein
